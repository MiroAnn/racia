Рация для горы

Работает через Firebase (Google) — надёжно на любой сети

Настройка — 2 минуты
1
Открой console.firebase.google.com → войди через Google
2
Нажми Add project → любое название → отключи аналитику → Create
3
В левом меню: Build → Realtime Database → Create database → Start in test mode → Enable
4
Скопируй URL из поля вверху (вида https://…firebaseio.com) → вставь ниже
5
⚙️ Project settings → General → скопируй Web API Key → вставь ниже
✓ Комната готова — поделись ссылкой
